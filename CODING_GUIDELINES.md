# 開発ガイドライン

このプロジェクトは、格闘ゲームのコンボを効率的に入力し、管理するためのツールです。

## 実行方法 (Electron版)

このアプリケーションをデスクトップ版として実行するには、以下の手順に従ってください。

1.  **依存関係のインストール:**
    ターミナルで以下のコマンドを実行し、必要なパッケージ（Electron）をインストールします。
    ```bash
    npm install
    ```

2.  **アプリケーションの起動:**
    インストールが完了したら、次のコマンドでアプリケーションを起動します。
    ```bash
    npm start
    ```

これにより、デスクトップアプリケーションとしてウィンドウが開き、ツールを使用できます。
